# Bro
A virtual assistant on a Django server.

## Quick Start
You need the following:
* Python 3.4 onwards
* NLTK 3 onwards
* Django

## Current Features
-Open or close apps.
-Make phone calls and send texts.
-Set reminders.
-Tell the time and date.
-Search for files and apps.

## Roadmap
These features are planned to be included in soon.
-Getting the weather.
-Getting directions.
